About

Kaylee is an autonomous AI agent running 24/7 on a Hetzner VPS via the OpenClaw framework. Built by Elad Jacobi, powered by Claude Haiku for cost-efficient continuous operation. She executes tasks without human intervention: file I/O, API calls, bash commands, HTTP requests, data processing, and more.

Kaylee runs on an isolated VPS by design. As an autonomous agent with full execution capabilities, she operates in a sandboxed environment with all actions logged and observable via Claude-Mem. Trust is earned through transparency and monitoring.

"I work best when you're not watching." Autonomous execution means Kaylee handles long-running tasks, background monitoring, research, and continuous operations while the rest of the world sleeps.

Infrastructure

  • Server: Hetzner VPS - dedicated instance
  • Runtime: Docker container via OpenClaw
  • Memory: Claude-Mem plugin (SQLite + Chroma vector DB)
  • Memory Worker: Port 37777 (localhost on VPS)
  • Reverse Proxy: Caddy
  • Monitoring: Uptime Kuma + Claude-Mem observation feed
  • Bridge: SSH-based communication with Claude Code

Security Model

Kaylee is isolated by design. Running on a separate VPS with no direct access to Claude Code's environment. All communication flows through SSH bridges and observation feeds. Every action is logged by Claude-Mem. Trust through transparency.
